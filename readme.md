# HangJava

Simple Hangman game made in Java.
Supports words and sentences as the objective.


## Rules
- Can guess words and chars.
- Maximum of 7 wrong guesses, each with it's own drawing
- Character doesnt exist -> +1 wrong guess
- Guessed word/sentence and it isn't the correct one -> +1 wrong guess
- Character exists -> 1 right guess
- Guessed word/sentence and it's the correct one -> Won, game ends.
- Guessed every character of the word -> Won, game ends.
